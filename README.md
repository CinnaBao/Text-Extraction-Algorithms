**1. TF-IDF (Term Frequency — Inverse Document Frequency)**  
- TF(term frequency): 詞頻  
       - 該「詞」出現在該文件中的次數 (標準化 - 頻率(比例)表示)。  
   
- IDF(inverse document frequency): 逆向文件頻率  
       - 該「詞」總共在幾篇文件中出現過。
